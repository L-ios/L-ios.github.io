26.删除排序数组中的重复项

本题主要是考察一个 有序数组 中的 O(1) 算法下的重复项过滤

  • 应用场景

    编码过程中，我们过滤重复项，主要是使用set，有此算法后，就可以抛弃set，进行重复项过滤

    • 限制条件
      • 有序数组
      • 非有序数组，重复项连续
  • 编码，解法，讲解
    int slow = 0;
for (int fast = 1; fast < nums.length; fast++) {
    if (nums[slow] != nums[fast]) {
        nums[++slow] = nums[fast];
    }
}
return slow + 1;

    本体主要是使用了快慢指针，利用快指针进行 有序重复数组 的遍历，慢指针进行原地更新
